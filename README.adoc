= Velkommen til Numsejuice

Verdens bedste opskriftbutik.

== Brunkager (100 stk)

  125g Smør
  70g  Lys sirup
  125g Brun farin
  75g  smuttede grofthakkede/hele mandler
  2 tsk kanel
  1 tsk stødt ingefær
  0,5 tsk stødt nellike
  0,5 tsk stødt allehånde
  0,5 tsk fintrevet økologisk abelsinskræl
  1 spsk abelsinsaft
